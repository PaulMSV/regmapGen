Register map
============

Автоматически сгенерировано с помощью программы regmapGen.

Сокращения
----------

.. list-table::
   :header-rows: 1

   * - Тип доступа
     - Описание
   * - rw
     - Read and Write
   * - rw1c
     - Read and Write 1 to Clear
   * - rw1s
     - Read and Write 1 to Set
   * - ro
     - Read Only
   * - roc
     - Read Only to Clear
   * - roll
     - Read Only / Latch Low
   * - rolh
     - Read Only / Latch High
   * - wo
     - Write only
   * - wosc
     - Write Only / Self Clear
